# Pratik Gadhave's Portfolio

Welcome to my personal portfolio website, featuring detailed case studies of my recent projects.

Built with **Vue 3**, **TypeScript**, and **Vite**. 
Interactive elements are powered by **GSAP**, **Lenis**, and **three.js**.

## Featured Projects

- **DocuMind**: Self-healing RAG engine for analyzing technical documents.
- **HiveMIND**: Multi-agent swarm architecture for complex query routing.
- **Codebase Intelligence**: AST-aware repository analysis tool with hybrid search.

## Scripts

| Command        | Description                          |
| -------------- | ------------------------------------ |
| `npm run dev`   | Dev server on port **3000** |
| `npm run build` | Compile and build to `dist/` |
| `npm run preview` | Serve the production build locally |
